JaRS - Job and Resource Scheduler

GitHub Repository
By Jacobus 'Coby' Coetsee

One of the things I came across while creating integration processes with older systems is that there were almost always a piece of the puzzle missing. Making paper-based processes digital sometimes had a requirement to send information to mobile devices, but with old systems lacking a way of scheduling jobs and communicating with mobile devices something  had to be done.. That is where JaRS comes in.

What is JaRS?

JaRS is a scheduling application that can form part of an integration process. It doesn't have to, it can be completley stand alone, but it was developed out of the need for a missing link in the integration process.
