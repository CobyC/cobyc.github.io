How to process files with python

The original post explains it very well!

I had to process a load of files on a raspberry pi. I knew about the os.listdir('path') function, but I was looking for something more efficient.

What if I had 1000 files and all I wanted to do is to check if there are any files in a folder. I didn't want to know that there were 1000 files or get a list of all the files in the folder.

The solution was os.scandir('path'), It allows you to iterate over the folders, so simply checking if there was 1 file was enough.

code or example


    def has_found_data_files(self):
        for file in os.scandir(self._app_config.record_file_to_path):
            if file is not None:
                return True
            else:
                return False
